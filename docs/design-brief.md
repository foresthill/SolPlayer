# SolPlayer デザインブリーフ（引き継ぎ）

> このドキュメントは、デザイン・フェーズを別会話（Fable想定）で進めるための引き継ぎメモです。
> 音声コア機能は一通り完成しており、ここからは **見た目（デザイン）とプレイヤー体験** が主題です。

## プロダクト概要

リアルタイム周波数変換機能付き音楽プレイヤー。A=440Hz を 432Hz / 444Hz などへ変換して再生できる。

## 現状（実装済み）

### 構成（モノレポ / Turborepo）
- `apps/web` — Next.js 15.5 (App Router) + React 19 + TypeScript + **Tailwind CSS v4**
- `apps/ios` — Swift（既存アプリ。`AVAudioUnitTimePitch` で周波数変換。本家挙動の参照）
- `packages/audio-core` — 音声処理。**SoundTouchJS** ベース
- `packages/shared-types` — 共通型

### 音声コア（完成済み・触らない前提でOK）
- **SoundTouchJS** でテンポとピッチを独立制御
  - 周波数変換 → `pitchSemitones`（テンポ維持）
  - 倍速 → `tempo`（ピッチ維持）
- 周波数変換の値は厳密式 `12*log2(target/base)`（iOSと一致：432→−31.7cent, 444→+15.7cent）
- 再生位置追従・クリックシークともに動作確認済み

### プレイヤーUI（現状は機能優先の素なダーク基調）
`apps/web/components/player/` に分割済み:
- `audio-player.tsx` — 全体コンテナ（ローカル音声ファイル選択で再生）
- `frequency-selector.tsx` — 440/432/444/437 プリセット＋カスタムHz(400–480)
- `playback-controls.tsx` — 再生/一時停止/停止
- `progress-bar.tsx` — シークバー
- `volume-control.tsx` — 音量
- `track-info.tsx` — 曲名/アーティスト
- ページ: `apps/web/app/page.tsx` が `<AudioPlayer />` をレンダリング

各コンポーネントの props インターフェースは維持したまま、**見た目だけ差し替え可能**な作りになっている。

## デザイン方針（ユーザー要望）

- **キーワード**: 先進的・透明感・グラスモーフィズム（フロステッドグラス）
- **背景**: 淡いパステルのグラデーション（ピンク → グリーン → ブルーが柔らかくにじむ）。ドリーミーでローファイな質感。余白は広め。
- **中心要素**: 白い半透明の再生ボタン（背景は透過）。コントロール類もガラス質感。
- **参照画像**: ユーザー提供（正方形＋縦長の2枚。ソフトなグラデーション見本）。中央に白い再生ボタンを置くイメージ。
- ライト/ダーク両対応が望ましい。

## 実装対象のUI要素

再生/一時停止/停止、プログレスバー（シーク）、周波数セレクタ、倍速スライダー、音量、トラック情報、ファイル選択ボタン。

## バックログ（デザインに織り込むと良い機能）

- **リピート**: 全曲リピート / 1曲リピート / シャッフル
- **プレイリスト管理**（Prisma schema に User/Playlist/Track は定義済み。DB層は未接続）
- 将来: YouTube / Spotify / Apple Music 連携、ビジュアライゼーション

## 進め方の選択肢

1. 参照画像から直接 Next.js(Tailwind v4) に実装する
2. デザイン仕様書 → モックアップ（claude design 等）で見た目を確定 → 実装する

推奨: **まず仕様＋モックで見た目を確定 → 器ができてからリピート等の機能を載せる**（素のUIに機能を足すと再デザイン時に作り直しになるため）。

## 技術メモ

- Tailwind CSS v4。グラスモーフィズムは `backdrop-blur` + 半透明背景 + 微細ボーダー + ソフトシャドウで表現。
- グラデ背景は CSS グラデーション（`radial-gradient` の重ね）で軽量に再現可能。画像を使う場合は `apps/web/public/` に配置。
- 既存の分割コンポーネントを活かし、共通の見た目（ガラスカード、トークン化した色）を定義してから各コントロールに適用すると統一感が出る。
